## 🧠 About Aia

Aia is an experimental virtual assistant project focused on creating a more dynamic and personal interaction experience.

Unlike a simple command-based assistant, Aia is designed around the idea of a digital entity capable of remembering information, developing emotional states, adapting responses and creating a continuous conversation experience.

The goal is to build an assistant that feels less like a tool and more like a presence that evolves over time.

---

## 💾 Memory System

Aia contains a persistent memory system responsible for storing important information about the user and previous interactions.

Currently, the memory system manages:

- User information
- Relationship values
- Emotional states
- Conversation history
- Events
- Preferences

The memory allows Aia to maintain continuity between sessions instead of starting from zero every time.

---

## 💬 Dialogue System

Aia uses a state-based dialogue system.

Instead of random responses, conversations are divided into different states, allowing the assistant to control the flow of interaction.

Examples:

- Greetings
- General questions
- Emotional conversations
- Future interaction states

Each dialogue state can contain different options and responses, making conversations more flexible and expandable.

---

## 🧠 Personality System

Aia has a personality framework designed to influence how she reacts and processes information.

Personality traits affect:

- Emotional reactions
- Response intensity
- Behavior tendencies
- Interaction style

The system is designed to grow with future updates, allowing more complex personality development.

---

## 🌙 Mood and Emotional System

Aia includes an emotional model that tracks different emotional values.

Current emotional categories include:

### Positive emotions

- Joy
- Happiness
- Affection
- Hope
- Enthusiasm
- Fun

### Negative emotions

- Anger
- Fear
- Anxiety
- Sadness
- Frustration
- Depression

### Obsessive emotions

- Jealousy
- Envy
- Obsession
- Anguish

These values influence how Aia behaves and responds during conversations.
